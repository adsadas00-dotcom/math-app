多項式の加法・減法

📖 解説
①
同類項（どうるいこう）とは
同類項 ＝ 文字の部分がまったく同じ項
📌 ポイント
文字の種類と次数が両方同じなら同類項！
係数が違っても文字部分が同じなら同類項！
同類項 ✅
3x と −5x
文字部分がどちらも x
同類項 ✅
2x² と 7x²
文字部分がどちらも x²
同類項でない ❌
3x と 2x²
x と x² で次数が違う
同類項でない ❌
4x と 4y
文字の種類が違う
②
同類項のまとめ方
同類項は係数どうしを計算してまとめられる！
📌 手順
同類項を見つけて → 係数を計算 → 文字はそのまま
式3x ＋ 5x
同類項を確認3x と 5x → 文字部分が両方 x → 同類項！
係数を計算（3＋5）× x ＝ 8x
まとめた結果 3x＋5x ＝ 8x
式4x²−x²＋2x−3x
x²の同類項4x² と −x² → （4−1）x² ＝ 3x²
xの同類項2x と −3x → （2−3）x ＝ −x
まとめた結果 3x²−x
③
多項式の加法（たし算）
多項式の加法 → かっこを外して同類項をまとめる
手順① かっこを外す 手順② 同類項を集める 手順③ まとめる
問題式（2x＋3）＋（4x−5）
①かっこ外す2x＋3＋4x−5
②同類項を集める2x＋4x＋3−5
③まとめる（2＋4）x＋（3−5）
答え：6x−2
④
多項式の減法（ひき算）
減法 → 引く式の符号を全部変えてからたす！
⚠️ 注意！符号の変化
−（A＋B） → −A−B 全部符号が変わる！
−（A−B） → −A＋B 引き算もプラスになる！
問題式（3x＋1）−（x−4）
①符号を変える 3x＋1−x＋4 ← −（x−4）＝−x＋4
②同類項を集める3x−x＋1＋4
③まとめる（3−1）x＋（1＋4）
答え：2x＋5
⚠️ よくある間違い
❌ 減法で符号を変え忘れる！
（3x＋1）−（x−4）＝ 3x＋1−x−4　← 間違い！
✅ −（x−4）は −x＋4 になる！　答え：2x＋5
引く式のかっこ内は全ての符号が変わる！
❌ 同類項でない項をまとめてしまう！
3x＋2x² ＝ 5x²　← 間違い！
✅ 3x（1次）と 2x²（2次）は同類項でない！　まとめられない。
文字の種類と次数が同じものだけまとめられる！
練　習　問　題
1
次の式を計算せよ。
（3x−2）＋（x＋5）
加法はかっこをそのまま外す。
かっこを外す：3x−2＋x＋5
同類項：xの項どうし、定数どうし
答え（途中式）
3x−2＋x＋5 ＝ （3＋1）x＋（−2＋5）
4x＋3
2
次の式を計算せよ。
（5a＋b）＋（−2a−3b）
かっこを外す：5a＋b−2a−3b
aの同類項、bの同類項をそれぞれまとめる。
答え（途中式）
a：5−2＝3　b：1−3＝−2
3a−2b
3
次の式を計算せよ。
（4x−3）−（2x＋1）
−（2x＋1）の符号を全部変える。
→ −2x−1　（プラスもマイナスになる！）
答え（途中式）
4x−3−2x−1 ＝ （4−2）x＋（−3−1）
2x−4
4
次の式を計算せよ。
（3x²＋2x−1）−（x²−x＋3）
−（x²−x＋3）の符号を全部変える。
→ −x²＋x−3
x²・x・定数の3種類の同類項をそれぞれまとめる。
答え（途中式）
x²：3−1＝2　x：2＋1＝3　定数：−1−3＝−4
2x²＋3x−4
5
次の式を計算せよ。
（2x²−3x＋4）＋（x²＋5x−2）
加法なのでかっこをそのまま外す。
x²・x・定数の3種類をそれぞれまとめる。
答え（途中式）
x²：2＋1＝3　x：−3＋5＝2　定数：4−2＝2
3x²＋2x＋2
6
次の式を計算せよ。
（4a−2b）−（a＋3b）＋（−2a＋b）
−（a＋3b）の符号を変える → −a−3b
3つのかっこを全部外してからa・bをまとめる。
答え（途中式）
a：4−1−2＝1　b：−2−3＋1＝−4
a−4b
🌟 追加問題でさらに定着！
+1
次の式を計算せよ。
かんたん
（2x＋7）＋（3x−4）
答え
5x＋3
+2
次の式を計算せよ。
かんたん
（6x−1）−（4x＋3）
答え
2x−4
+3
次の式を計算せよ。
かんたん
（−3a＋5b）＋（2a−b）
答え
−a＋4b
+4
次の式を計算せよ。
中級
（x²−3x＋2）−（2x²＋x−5）
−（2x²＋x−5）の符号を全部変える。
→ −2x²−x＋5
x²・x・定数の3種類を別々にまとめる。
答え（途中式）
x²：1−2＝−1　x：−3−1＝−4　定数：2＋5＝7
−x²−4x＋7
+5
次の式を計算せよ。
中級
（3a＋b）−（a−2b）＋（−a＋3b）
−（a−2b）の符号を変える → −a＋2b
（−a＋3b）はそのまま外す。
全部並べてa・bをまとめる。
答え（途中式）
a：3−1−1＝1　b：1＋2＋3＝6
a＋6b
