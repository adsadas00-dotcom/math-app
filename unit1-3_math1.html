解説・重要ポイント
重 要 ！
① 絶対値とは

絶対値とは、数直線上で0からその数までの距離のことです。
記号は両側に縦棒をつけて表します（例：|−3| ＝ 3）

絶対値の求め方
符号をとりのぞいた数（プラス・マイナスを無視した数）が絶対値
例：|−3| ＝ 3　　|+4| ＝ 4　　|0| ＝ 0
重 要 ！
② 絶対値は必ず0以上

絶対値は「距離」なので、マイナスにはなりません。かならず 0 以上になります。

+9 の絶対値
0から右に9の距離
9
−6 の絶対値
0から左に6の距離
6
0 の絶対値
0から0の距離
0
✅ 絶対値 ＝ 符号を外した数（マイナスがついていても外してしまえばOK！）
重 要 ！
③ 絶対値が等しい2つの数

絶対値が同じ数は、＋と−の2つ存在します（0だけは1つ）。
0から同じ距離にある2点は、0の両側に1つずつあるから。

絶対値が6の数は？
+6 と −6 の2つ
絶対値が2.5の数は？
+2.5 と −2.5 の2つ
絶対値が0の数は？
0 の1つだけ
覚え方
絶対値が n の数 → +n と −n の2つ（ただし n=0 のときは 0 だけ）
練習問題
練　習　問　題
(1)
次の絶対値を求めよ。
① +9　　② −6　　③ 0　　④ −2.5
① 9　　② 6　　③ 0　　④ 2.5
(2)
絶対値が5の数をすべて答えよ。
0から距離5の数は、右と左に1つずつある。
+5 と −5
(3)
絶対値が大きいのはどちらか答えよ。
　−8 と +5
それぞれの絶対値を求めてから比べよう。
−8の絶対値は 8、+5の絶対値は 5
8 ＞ 5 なので、絶対値が大きいのは −8
🔥 もっと練習しよう！
(4)
次の絶対値を求めよ。かんたん
① −100　　② +0.3　　③ −12
① 100　　② 0.3　　③ 1/2（2分の1）
(5)
絶対値が3の数と絶対値が7の数をすべて答えよ。かんたん
絶対値が3の数：+3 と −3
絶対値が7の数：+7 と −7
(6)
次の文が正しければ○、間違っていれば✕と答えよ。かんたん
① 絶対値は必ず正の数になる。
② −7 の絶対値は −7 である。
③ 絶対値が等しい2つの数は必ず存在する。
① ✕（0の絶対値は0なので、正の数とは限らない。0以上が正しい）
② ✕（−7の絶対値は7。絶対値は負にならない）
③ ✕（絶対値が0の数は0だけ。0に対応する「もう1つの数」はない）
(7)
−4、+6、−2、+1 を絶対値の大きい順に並べよ。中級
まず各数の絶対値を求めてみよう。−4→4、+6→6、−2→2、+1→1
各絶対値：|+6|=6、|−4|=4、|−2|=2、|+1|=1
絶対値の大きい順：+6、−4、−2、+1
(8)
絶対値が4より小さい整数をすべて答えよ。中級
絶対値が4より小さい → 0からの距離が4未満。数直線で0の近くにある整数を探そう。
−3、−2、−1、0、+1、+2、+3（7個）
